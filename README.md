# 本ツールについて

AI を用いて、ノードベースのテクスチャ作成ツールをどこまで形にできるかを試すために開発したプロジェクトです。  
実験的な性質が強く、現時点では十分な動作確認を行っていないため、正常な動作は保証していません。



## 概要

本ツールは、AI を活用した開発によってノードベースのテクスチャ作成ツールの実現可能性を検証するために制作しました。  
完成された製品ではなく試作段階のプロジェクトであり、機能不足や不安定な挙動、予期しない不具合が含まれる可能性があります。



## 特徴

- ノードベースのテクスチャ作成ツールの試作
- AI を活用した実装および検証
- 実験的な構成・設計を含むプロジェクト
- テクスチャ制作ツールの可能性を探るための検証用途



## 動作環境

- OS: Windows 10 / 11
- 配布形式: exe
- 備考: 実行環境によっては正常に動作しない場合があります



## インストール

本ツールは exe 形式での配布を想定しています。  
Python のインストールは不要です。

1. Release から zip ファイルをダウンロードします
2. zip ファイルを任意の場所に展開します
3. `Snaline.exe` を実行します



## 使い方

1. `Snaline.exe` を起動します
2. 必要に応じてノードを追加します
3. 各種設定を行い、テクスチャ作成や検証を行います

※ 実験的なツールのため、機能や操作方法は今後変更される可能性があります。



## 配布に関する注意

- exe 配布の性質上、環境によっては Windows Defender や SmartScreen による警告が表示される場合があります
- 配布ファイルの利用は自己責任でお願いします

## AI × ツール開発に関する所感

権利面などの課題が整理・解決されるのであれば、AI を開発に取り入れる意義は大きいと感じています。  
本ツールは、業界経験3年のデザイナーが約50時間で開発しました。

このことから、TA が AI を活用すれば、1人月程度でも既存の DCC ツールに近い機能を備えた自社ツールを開発できる可能性があると考えています。  
そのような取り組みが実現すれば、制作環境に合わせた柔軟な機能設計が可能になるだけでなく、ライセンスコストの削減にもつながるかもしれません。
