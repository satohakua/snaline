## 概要

本ツールは、AI を活用した開発によってノードベースのテクスチャ作成ツールの実現可能性を検証するために制作しました。  
完成された製品ではなく試作段階のプロジェクトであり、機能不足や不安定な挙動、予期しない不具合が含まれる可能性があります。

<br>

## 動作環境

- OS: Windows 10 / 11
- 配布形式: exe
- 備考: 実行環境によっては正常に動作しない場合があります

<br>

## インストール

本ツールは exe 形式のため、Python のインストールは不要です。

1. Release から zip ファイルをダウンロード
2. zip ファイルを任意の場所に展開
3. `Snaline.exe` を実行

<br>

## 使い方

### 1. 画面の見方

- 左: ノード一覧
- 中央: ノードを組むキャンバス
- 右: 選択中ノードのプレビュー
- 上: パンくず。サブグラフを開いたときに階層移動できます

### 2. ノードを追加する

- 左のノード一覧をダブルクリックすると追加できます
- 左のノード一覧からキャンバスへドラッグ&ドロップでも追加できます
- `Tab` でノード検索を開いて追加できます

### 3. ノードをつなぐ

- ノード右側の出力ソケットから、別ノード左側の入力ソケットへドラッグします
- 画像系は画像入力へ、数値系は数値入力へつなぎます
- 選択したノードは右側プレビューで結果を確認できます

### 4. パラメーターを変える

- ノード下部のパラメーター欄を直接編集します
- 数値はキーボード入力で変更できます
- 数値欄は右ドラッグでも調整できます
- 標準ノードの内部はロックされています。`No editable` が出ている場所は編集できません

### 5. キャンバス操作

- 中ドラッグ: 画面移動
- マウスホイール: ズーム
- ノードドラッグ: ノード移動
- `Delete` / `Backspace`: 選択ノード削除

### 6. 保存と書き出し

- `File > Save` で `.texnode` として保存します
- `File > Open` で `.texnode` を開きます
- `File > Export` で `PNG / JPG / TIFF` に書き出します
- 書き出しは `Output` ノードの結果が対象です

### 7. よく使うショートカット

- `Ctrl+S`: 保存
- `Ctrl+Z`: 元に戻す
- `Ctrl+Shift+Z`: やり直し
- `Ctrl+C`: ノードコピー
- `Ctrl+V`: ノード貼り付け
- `Ctrl+Alt+C`: パラメーターコピー
- `Ctrl+Alt+V`: パラメーター貼り付け
- `Ctrl+Alt+R`: パラメーター初期化
- `Tab`: ノード検索
